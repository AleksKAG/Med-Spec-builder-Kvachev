# Спецификации технологического оборудования медицинской организации (Pet Project)

Простое веб-приложение для выбора кабинетов и автоматического формирования спецификации медицинского оборудования из шаблона Excel в соответствии с Приказами Министерства Здравоохранения РФ.
Идеально подходит для медицинских учреждений, проектировщиков и закупщиков.

## Цель проекта
- Программа формирует сводную спецификацию медицинского оборудования с агрегацией количеств по экспликации помещений
- Экспортировать результат в Excel

## Технологии

- **Backend**: Go 1.22 + Gin + GORM + PostgreSQL + Excelize
- **Frontend**: Vue 3 + Vite + Axios + Tailwind CSS (опционально)
- **DevOps**: Docker + docker-compose
- **Исходные данные**: Excel-файл с 1500+ строками (разделён по листам: «Кабинеты взрослые», «Кабинеты детство», «Стерилизация» и др.)

## Запуск проекта

### Требования
- Docker и Docker Compose
- Excel-файл `Приложение №1` в папке `init-data/`

### Шаги

1. Склонируйте проект:
   git clone https://github.com/yourname/spec-builder.git
   cd spec-builder
2. Убедитесь, что файл Excel лежит в:
   init-data/Приложение №1
3. Запустите:
   docker-compose up --build
4. Откройте в браузере:
Фронтенд: http://localhost:5173
Backend API: http://localhost:8080/api/cabinets

При первом запуске данные автоматически импортируются в PostgreSQL (~10–30 сек).
